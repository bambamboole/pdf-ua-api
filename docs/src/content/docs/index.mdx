---

PDF UA API converts HTML to **PDF/A-3a** documents with full **PDF/UA** accessibility,
validates PDFs with veraPDF, and renders HTML to images — all over a simple HTTP API.

## Highlights

- HTML → accessible PDF/A-3a conversion
- Built-in PDF/A & PDF/UA validation (veraPDF)
- Bundled open-source fonts, full CSS 2.1 styling, automatic table pagination
- Visual [template builder](/template-builder/) and a live [playground](/playground/)

Start with the [Quick Start](/getting-started/quick-start/) or browse the
[API Reference](/api/).
